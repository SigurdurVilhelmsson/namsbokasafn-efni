# Lykiljöfnur

$$q = c \times m \times \Delta T = c \times m \times (T_{\text{lok}} - T_{\text{upphafs}})$$

þar sem:
- *q* = varmi (J)
- *c* = eðlisvarmi (J/g °C)
- *m* = massi (g)
- Δ*T* = hitastigsbreyting (°C)
