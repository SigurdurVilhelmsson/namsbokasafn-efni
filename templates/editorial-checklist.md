## Pass 1: Linguistic Review

### Before Starting

- [ ] Received .docx file from translator
- [ ] Have access to terminology glossary (`glossary/terminology-en-is.csv`)
- [ ] Have access to terminology guide (`docs/terminology.md`)
- [ ] Understand: **NO localization changes in this pass**
- [ ] Word version is compatible (Word 2016+ recommended)

### Setup

- [ ] Enable Track Changes in Word (Review → Track Changes)
- [ ] Verify your name is set correctly in Word preferences
- [ ] Save a backup copy before starting

### Review Process

For each section:

- [ ] **Read through once** without making changes
- [ ] **Review for grammar and spelling**
  - [ ] Subject-verb agreement
  - [ ] Correct case usage
  - [ ] Spelling errors
  - [ ] Punctuation
- [ ] **Review for natural Icelandic**
  - [ ] Sentence structure flows naturally
  - [ ] Word order is idiomatic
  - [ ] No awkward direct translations
- [ ] **Check terminology**
  - [ ] All terms match glossary
  - [ ] Terms used consistently throughout
  - [ ] Flag any uncertain terms with comments
- [ ] **Verify technical accuracy**
  - [ ] Scientific facts preserved
  - [ ] Equations correct
  - [ ] Numbers unchanged (no conversions!)

### What to Look For

**Grammar Issues:**
- [ ] Incorrect declensions
- [ ] Wrong article usage
- [ ] Verb tense consistency
- [ ] Preposition errors

**Style Issues:**
- [ ] Overly literal translations
- [ ] Anglicisms that have Icelandic equivalents
- [ ] Inconsistent register
- [ ] Unclear antecedents

**DO NOT Change:**
- [ ] Units (miles, Fahrenheit, etc.) - keep as source
- [ ] Cultural references - keep as source
- [ ] Examples - keep as source
- [ ] Formatting

### Adding Comments

Use comments for:
- [ ] Questions about meaning
- [ ] Terminology suggestions
- [ ] Flagged issues for discussion
- [ ] Notes about uncertainty

Mark urgency with tags:
- `[QUESTION]` - Need clarification
- `[URGENT]` - Important issue
- `[DISCUSS]` - Needs team discussion
- `[TERM]` - Terminology question

### Completion Checklist

- [ ] All sections reviewed
- [ ] Track Changes shows all edits
- [ ] Comments added for questions
- [ ] Terminology questions documented
- [ ] No localization changes made
- [ ] File saved as: `[section]-pass1-[initials].docx`

## Pass 2: Localization Review

### Before Starting

- [ ] Pass 1 is complete and approved
- [ ] TM has been updated with Pass 1 edits
- [ ] Have access to localization log template
- [ ] Have access to unit conversion reference
- [ ] Understand localization guidelines

### Setup

- [ ] Start from faithful translation in `03-faithful/docx/ch##/`
- [ ] Create localization log from template (`templates/localization-log.md`)
- [ ] Open unit conversion reference

### Review Process

For each section:

- [ ] **Identify localization opportunities**
  - [ ] Imperial units that need conversion
  - [ ] American cultural references
  - [ ] Examples that could use local context
  - [ ] Places where extended exercises would help

- [ ] **Convert units**
  - [ ] Miles → kilometers
  - [ ] Feet/inches → meters/centimeters
  - [ ] Fahrenheit → Celsius
  - [ ] Pounds → kilograms
  - [ ] Gallons → liters
  - [ ] Verify all calculations are correct

- [ ] **Adapt cultural references**
  - [ ] American holidays → Icelandic equivalents
  - [ ] American geography → Icelandic examples (where appropriate)
  - [ ] American institutions → Icelandic equivalents (where appropriate)

- [ ] **Add Icelandic context** (where it enhances understanding)
  - [ ] Geothermal energy connections
  - [ ] Fishing industry applications
  - [ ] Icelandic geology/geography
  - [ ] Local environmental issues

- [ ] **Add extended exercises** (where beneficial)
  - [ ] Practice problems with SI units
  - [ ] Problems with Icelandic context
  - [ ] Additional worked examples

### Documentation

For EVERY change:
- [ ] Record in localization log
- [ ] Note original text
- [ ] Note new text
- [ ] Include rationale

### Verification

- [ ] All unit conversions double-checked
- [ ] Converted calculations verified
- [ ] Scientific accuracy maintained
- [ ] No information lost
- [ ] Additions enhance (not distract from) learning

- [ ] Scientific accuracy confirmed
