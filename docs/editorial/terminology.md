# Terminology Standards

This document establishes terminology standards for Icelandic translations of OpenStax textbooks.

## Why Consistent Terminology Matters

Consistent terminology is crucial for:
- **Clarity** - Students encounter the same terms throughout
- **Searchability** - Consistent terms make content findable
- **Standards** - Aligns with established Icelandic scientific vocabulary
- **Quality** - Inconsistent terminology indicates poor translation
- **Reusability** - TM and glossary become valuable for other projects

## Principles for Choosing Icelandic Terms

### 1. Prefer Established Terms

Always use established Icelandic terms when they exist. Check sources in this order:

1. **Íðorðabankinn** (Icelandic Term Bank)
   https://idord.arnastofnun.is/

2. **Orðabanki Háskóla Íslands**
   https://ordabanki.hi.is/

3. **Established textbooks** - Chemistry and biology textbooks used in Icelandic schools

4. **Project glossary** - `glossary/terminology-en-is.csv`

### 2. Follow Icelandic Naming Conventions

Icelandic has strong conventions for scientific terminology:

| Pattern | Example |
|---------|---------|
| -eind (particle) | rafeind (electron), róteind (proton) |
| -efni (substance/element) | frumefni (element), leysiefni (solvent) |
| -tengi (bond/connection) | efnatengi (chemical bond) |
| -hvarf (transformation) | efnahvarf (chemical reaction) |

### 3. Consider the Target Audience

- Secondary school students (16-20 years)
- Basic scientific literacy assumed
- Terms should be understandable without excessive explanation
- Prefer transparent compounds over opaque borrowings when possible

### 4. Maintain Consistency

Once a term is chosen:
- Use it consistently throughout
- Add it to the glossary
- Document the decision and source

## Adding New Terms

When you encounter a term not in this list:

### Step 1: Research
1. Check Íðorðabankinn
2. Check Orðabanki HÍ
3. Check Icelandic textbooks
4. Check academic papers

### Step 2: Document
If found, add to the glossary with:
- English term
- Icelandic term
- Category
- Source reference
- Any notes about usage

### Step 3: Discuss if Needed
If not found or unclear:
- Discuss with project lead
- Consider existing patterns
- Document the reasoning

### Step 4: Update Glossary
Add the term to `glossary/terminology-en-is.csv`

## Terminology Disputes

If there is disagreement about a term:

1. **Check official sources** - Íðorðabankinn is authoritative
2. **Consider context** - What's used in Icelandic education?
3. **Prefer established terms** - Even if less "accurate"
4. **Consider the audience** - Will students understand?
5. **Document the decision** - Include reasoning in glossary

## Format for Terminology Entries

### In Documentation

| English | Icelandic | Notes | Source |
|---------|-----------|-------|--------|
| term | íslenskt hugtak | Usage context | Reference |

### In CSV Glossary

```csv
english,icelandic,category,notes,source,status
atom,atóm,fundamental,,"Icelandic naming convention",approved
```

Status options:
- `approved` - Verified and approved
- `proposed` - Proposed, awaiting review
- `deprecated` - No longer used
